    抓取Linux公社资料

    之前搜索资料时发现一个比较好的网站`Linux公社’ ,类似FTP服务的形式提供很多pdf资料的下载.但是内容都是按时间分类的 ,我就想着把列表扒下来 ,需要的时候搜索一下书名 ,然后获取下载连接 .( 书名索引 )

    • 爬取Linux公社页面 ,分析链接 ( 文件夹链接/文件链接 )
    • 文件链接使用 name:url直接存储 ; 文件夹链接放入待抓取队列
    • 持续分析待抓取队列中的url
    • 生成索引文件
    • 提供指定目录下载的功能
